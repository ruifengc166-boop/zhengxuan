# 政宣智作高保真原型说明（V2）

打开 `index.html` 即可离线体验，无需安装依赖或启动服务。

## 当前版本定位

当前版本采用“个人创作 + 用户自审”模式，不包含团队成员、任务分派、审批人、待办箱和多级签发。团队审批将在后续版本单独设计。

主流程为：

资料与脚本 → 全片分镜图 → 全片视频镜头 → 剪辑与声音 → 成片优化 → 自动扫描 → 用户发布自检 → 正式导出

## 核心能力

- 全局导航聚焦创作：首页、创作项目、素材与资料、模板中心、自检发布、模型与 API、额度与计费；
- 项目内部采用“创作制作—剪辑预览—自检发布”三个主工作区；
- 支持验证码登录、密码登录、组织注册、统一身份认证入口和退出登录；
- 支持平台托管 API、单位自有 API、用户自带 API Key；
- 文生文、文生图、文生视频均可选择供应商和模型；
- AI 根据脚本动态建议镜头数量，用户可新增、删除、拆分、合并和调整顺序，不预设固定镜头数；
- 每个镜头均包含描述、正向 Prompt、负面约束、多图参考、模型、比例和生成数量；
- 批量生图只生成候选，不自动选片或定稿，必须由用户逐镜头确认；
- 候选图支持再次抽卡、局部重绘、扩图、作为参考图及基于候选继续编辑；
- 候选区域默认只显示“当前候选图”，历史轮次通过“查看生成历史”进入，避免脱离上下文显示“第 2 轮”；
- 镜头图定稿后可重新编辑，但会提示对应的下游视频版本失效；
- 创作页取消独立 AI 助手侧栏，工作台全宽展示，操作提示直接放在对应功能附近；
- 多参考可组合人物、场景、图片、视频、音频、动作、运镜、风格和特效；
- 支持文生视频、首帧、首尾帧、全能多参考、多镜头、续写和视频编辑；
- 后期支持口型、LUT、重新打光、超分、补帧、防抖和去噪；
- “剪辑与声音”提供完整成片播放器、视频/旁白/音乐/字幕时间线、音色试听、语速音量调整和声音预览版本确认；
- “画质优化”提供优化前后拖动对比、逐镜头任务状态、失败镜头单独重试、应用优化版本和切回原始版本；
- 未确认剪辑声音版本不能进入画质优化，未应用优化版本不能进入发布自检；
- 平台 API 具备钱包、充值、参考单价、项目预算、费用归集和调用明细。

## 发布自检

系统先自动扫描文字、画面、声音、来源、授权和 AI 标识，再由当前用户完成四项确认：

1. 内容、事实、字幕、数字和发布口径无误；
2. 肖像、声音、图片、字体和音乐具有发布授权；
3. 不含涉密信息、敏感个人信息和禁止公开的内部资料；
4. 保留显式、隐式 AI 标识并符合目标渠道要求。

四项全部勾选后，“确认自检并解锁导出”按钮才可使用。确认记录与导出版本留痕，但不代表其他人员或单位完成了专业审批。

## 正式导出

- 可选择 16:9 横屏、9:16 竖屏或无字幕洁净版；
- 支持 1080P H.264、4K H.265 和 ProRes MOV；
- 支持烧录字幕、独立 SRT 或无字幕；
- 默认写入显式 AI 标识和文件元数据隐式标识；
- 可下载 MP4 视频，或包含视频、字幕、封面和版权合规材料的 ZIP 发布包。

## 演示主线

1. 登录并进入“创作项目”；
2. 打开项目，完成全部镜头的图像 Prompt、分镜候选和锁定；
3. 完成全部镜头的视频 Prompt、多参考生成、抽卡和锁定；
4. 在剪辑预览中调整旁白、音乐、字幕、片头片尾和 AI 标识；
5. 进入“自检发布”，查看自动扫描结果；
6. 逐项完成本人确认并解锁正式导出；
7. 选择交付版本，生成并下载视频或完整发布包。

## 后续版本

团队模式将在后续版本增加，届时再设计组织角色、项目协作者、审批人待办、多级审批、退回指定环节、OA 通知和发布签发。本版本不展示这些能力。

## 原型范围

该版本用于产品汇报、客户访谈与研发需求澄清，不连接真实模型、扫描服务、业务数据库或下载服务。页面中的模型名称、费用、扫描结果和下载行为均为演示数据。
